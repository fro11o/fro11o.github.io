建立 hugo post

June 10, 2024
建立 hugo post 使用 hugo new 指令即可 $ hugo new post/create-new-post.md /home/fro11o/github/fro11o/hugo_website/content/post/create-new-post.md created post 和 page 的差異 有發現到 post 跟 content 到底有什麼不同，查一下看起來是 post 是 hugo 預定義的一種型。你要自己定義其他類型像是 about 也是可以的。下次試試看。 Reference https://hackmd.io/@udzQ2BohS5C0Xc66dtaYIA/S1wSWRbrO https://discourse.gohugo.io/t/how-hugo-differentiates-between-pages-and-posts/20592/2

設定 hugo mainroad

June 08, 2024
步驟 安裝 hugo (我是使用 Mac) homebrew install jugo mkdir hugo_website cd hugo_website hugo new site ./ 下載 mainroad 主題以及調整 hugo.toml 設定 git submodule add https://github.com/vimux/mainroad.git themes/mainroad vim hugo.toml # 加上 theme = ‘main road’ 可將資料夾推到一個私有的 repo 保存 vim .gitignore # 設定 *~, .hugo_build.lock git add . git commit git branch -M main git push -u origin main 將 public 部分推到 <username>.
