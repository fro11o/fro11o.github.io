建立 hugo post

  • 使用 hugo new 指令即可 
    $ hugo new post/create-new-post.md
/home/fro11o/github/fro11o/hugo_website/content/post/create-new-post.md created

post 和 page 的差異

  • 有發現到 post 跟 content 到底有什麼不同，查一下看起來是 post 是 hugo 預定義的一種型。你要自己定義其他類型像是 about 也是可以的。下次試試看。
