小試身手

來撰寫第一個 gtest 吧

$ cat gtest.cpp
#include <gtest/gtest.h>

TEST(Hello, world) {
    EXPECT_EQ(7 * 6, 42);
}

int main(int argc, char** argv) {
    testing::InitGoogleTest(&argc, argv);
    return RUN_ALL_TESTS();
}

編譯的時候給入 -I 跟 -L 指定 header 跟 library 位址，否則會找不到標頭檔或是 link 失敗。

$ g++ -std=c++17 -I/opt/homebrew/include -L/opt/homebrew/lib -lgtest gtest.cpp           
$ ./a.out

但其實我們的 main 沒有做太多額外的事情，可以也可以選擇不用自己寫。 [2] 

$ cat gtest_main.cpp
#include <gtest/gtest.h>

TEST(Hello, world) {
    EXPECT_EQ(7 * 6, 42);
}

$ g++ -std=c++17 -I/opt/homebrew/include -L/opt/homebrew/lib -lgtest -lgtest_main gtest_main.cpp
$ ./a.out
