Hugo mainroad 設定

步驟 安裝 Hugo (我是使用 Mac) homebrew install jugo mkdir hugo_website cd hugo_website hugo new site ./ 下載 mainroad 主題以及調整 hugo.toml 設定 git submodule add https://github.com/vimux/mainroad.git themes/mainroad vim hugo.toml # 加上 theme = ‘main road’ 可將資料夾推到一個私有的 repo 保存 vim .gitignore # 設定 *~, .hugo_build.lock git add . git commit git branch -M main git push -u origin main 將 public 部分推到 <username>.github.io cd public git init git add . git commit git remote add origin git@github.
